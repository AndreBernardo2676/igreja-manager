
# Church Manager System

Sistema web para gerenciamento de igrejas, permitindo controle de membros, eventos e informações administrativas em uma interface simples e moderna.

## 📌 Sobre o projeto

O **Church Manager System** foi desenvolvido com o objetivo de auxiliar igrejas pequenas e médias na organização de suas atividades administrativas.

O sistema permite centralizar informações importantes como cadastro de membros, controle de eventos e gerenciamento básico da comunidade.

Este projeto foi desenvolvido como parte de estudos em desenvolvimento de software e demonstra o uso de ferramentas modernas de desenvolvimento, versionamento e deploy.

---

## 🚀 Funcionalidades

* Cadastro e gerenciamento de membros
* Sistema de autenticação de usuários
* Gerenciamento de eventos da igreja
* Interface web simples e intuitiva
* Banco de dados integrado
* Sistema pronto para deploy online

## 💻 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/AndreBernardo2676/Church-Manager-system.git
```

Entre na pasta do projeto:

```bash
cd Church-Manager-system
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm start
```

O sistema ficará disponível em:

```
http://localhost:3000
```
